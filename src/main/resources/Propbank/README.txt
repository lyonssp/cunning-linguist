============================
MASC PropBank Original files
============================

This distribution includes the following:

 Propbank-orig
 -------------
 PropBank annotations in the original format produced by the Propbank
 project for 88,530 words of MASC data.

 Penn_Treebank-orig
 ------------------
 Penn Treebank syntax annotations in the PTB original bracketed format 
 for the same 88,530 words of MASC data.

 MASC1_textfiles
 ---------------
 The 88,530 word subset of MASC texts to which the Propbank and Penn
 Treebank annotations are applied. Note that the Penn Treebank
 annotations include the actual text inline, so the inclusion of the
 unannotated text files here is simply for reference.

These Penn Treebank syntax and Propbank annotations are also distributed
by the ANC project in the standoff GrAF format used for all MASC
annotations. The ANC project is not responsible for any errors in the
original data.
